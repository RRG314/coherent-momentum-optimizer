# Changelog

## 0.1.0

- Created standalone repository for the Magneto-Hamiltonian optimizer family.
- Promoted `MagnetoHamiltonianAdam` as the mainline branch.
- Included `HamiltonianAdamReal` as the documented base reference.
- Added accepted result snapshots, energy-test summaries, README tables, figures, literature references, and reproduction scripts.
- Narrowed the public package surface so the repo presents one optimizer family cleanly.
