# Changelog

## 0.1.0

- Created standalone repository for the Coherent Momentum optimizer family.
- Promoted `CoherentMomentumOptimizer` as the mainline branch.
- Included `CoherentMomentumRealBaseline` as the documented base reference.
- Added accepted result snapshots, energy-test summaries, README tables, figures, literature references, and reproduction scripts.
- Narrowed the public package surface so the repo presents one optimizer family cleanly.
